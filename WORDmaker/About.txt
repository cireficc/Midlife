WORDmaker v.01 
by Daniel de la Rosa (aka. RadioSpace)

Purpose: 
	To enter data of type Language.Word into an XML file for use by the Midlife Language.ExtendedDictionary. this program is not intented for end users of the dictionary.
	its intention is for use by the developer of Midlife and its dictionary.

Objective:
    to effeicntly add Language.Word objects to an xml file that will grow to between 50,000 and 100,000 entries. should also be able to display list of entered words, check for duplicates,
	and later browse the word details in a dialogbox.

v.01: started 2/4/14. simple UI with collapsable textboxes for conjuctiontables
